## Overview

AuthentiScan is a machine learning system that detects whether an Instagram caption was written by a human or generated by an AI. It also identifies **which AI** generated it, whether ChatGPT, Gemini, Copilot, or Perplexity.

Built as an independent research project, AuthentiScan addresses a gap in existing AI detection tools: most are designed for long-form academic text and fail on short informal content under 150 words.

## How It Works

AuthentiScan uses a three-component hybrid feature fusion approach:

**1. TF-IDF Features (5,000 features)**
Captures vocabulary-level patterns. AI captions use formal structured language while human captions use slang, contractions, and casual phrasing.

**2. SBERT Embeddings (384 dimensions)**
Sentence-level semantic embeddings from `all-MiniLM-L6-v2` capturing stylistic patterns at the caption level.

**3. Handcrafted Linguistic Indicators (7 features)**
Emoji presence, em dash usage, ellipsis, AI-associated starter verbs, buzzwords, word count, character count.

**PSO Optimization**
Rather than combining features with equal weights, Particle Swarm Optimization learns the optimal weight for each feature component, maximizing classification accuracy through global search over the weight space.

## Using the Project Files

**Merge or rebuild the dataset:**
```bash
python src/merge_dataset.py
```
Reads `data/raw/manual_captions.csv` and `data/raw/ai_captions.csv`, computes all indicator features, balances and shuffles the dataset, and saves the final output to `data/processed/dataset.csv`.

**Run the full ML pipeline:**

Open `notebooks/authentiscan_development.ipynb` directly in VS Code and select the `venv` Python interpreter as the kernel when prompted. Run cells sequentially. The notebook covers:

- Data loading and exploration
- EDA visualizations
- Text cleaning and preprocessing
- TF-IDF and SBERT feature extraction
- Handcrafted feature computation
- PSO optimization and weight search
- Baseline model comparison (LR, SVM, RF, Gradient Boosting)
- Final PSO model evaluation and confusion matrices
- Phase 2 AI source attribution experiment
- Ablation study

All results and plots are saved automatically to the `results/` folder.
